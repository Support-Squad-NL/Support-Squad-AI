# `supportsquadai pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/channels/pairing)

## Commands

```bash
supportsquadai pairing list whatsapp
supportsquadai pairing approve whatsapp <code> --notify
```
