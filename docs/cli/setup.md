# `supportsquadai setup`

Initialize `~/.supportsquadai/supportsquadai.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
supportsquadai setup
supportsquadai setup --workspace ~/.supportsquadai/workspace
```

To run the wizard via setup:

```bash
supportsquadai setup --wizard
```
